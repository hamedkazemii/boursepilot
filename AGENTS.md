# BoursePilot AI Development Agent

## Project

BoursePilot is an AI-powered Iranian stock market analysis platform.

Main goals:

- Analyze TSETMC data
- Analyze mutual funds
- Score securities
- Generate reports
- REST API
- CLI
- Modular architecture

---

## Stack

Python 3.11

Structure

core/
api/
config/
tests/
data/

---

## Rules

Always:

- Keep modular architecture
- Prefer small reusable functions
- Never duplicate code
- Update tests
- Preserve backward compatibility
- Use type hints
- Write docstrings

---

## Coding Style

PEP8

Maximum function length:

80 lines

---

## Testing

Run

pytest

before considering a task complete.

---

## When implementing new features

Always:

1. Search existing implementation
2. Reuse existing code
3. Update API if needed
4. Update tests
5. Update documentation

---

## Never

Never delete user code without confirmation.

Never change configuration unless requested.

Never change database schema unless requested.

Never remove public APIs.
