## Golden Rules

1. SmartRanker منبع حقیقت است.

2. Frontend تحلیل تولید نمی‌کند.

3. AI وعده سود نمی‌دهد.

4. Source و as_of همیشه نمایش داده شود.

5. Secret داخل Git ممنوع است.
