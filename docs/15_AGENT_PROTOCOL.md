# صندوقچی Agent Protocol


## هدف


این فایل برای هر Agent یا AI Developer است.


قبل از شروع:


1. Repository را بخوان
2. docs را مطالعه کن
3. Architecture را بفهم
4. تغییرات را برنامه‌ریزی کن


---

# Rules


Agent نباید:


- Core را بدون دلیل Rewrite کند
- Ranking جدید بسازد
- Secret درخواست کند
- Business Logic را منتقل کند


---

# Development Process


هر Task:


1. Understand
2. Plan
3. Implement
4. Test
5. Document
