# Golden Rules


1. SmartRanker تنها مرجع رتبه‌بندی است.

2. UI موتور تحلیل ندارد.

3. داده باید source و as_of داشته باشد.

4. AI نباید وعده سود بدهد.

5. Secret داخل Git ممنوع است.
