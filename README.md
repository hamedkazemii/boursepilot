# 🟢 صندوقچی (Sandoghchi / BoursePilot)

## AI Assistant for Iranian Investment Funds


صندوقچی یک دستیار هوشمند تحلیل صندوق‌های سرمایه‌گذاری ایران است.

قابلیت‌ها:

- دریافت داده بازار
- تحلیل تاریخچه
- اندیکاتورهای تکنیکال
- رتبه‌بندی هوشمند صندوق‌ها
- تحلیل پرتفو
- مشاور هوشمند AI
